## Project Overview
  ### Website Flow
  When users enter the website, they will be greeted with the home page that has 3 menus:
  - About
  - Menu
  - Contact

  In the menu, there are also 3 submenus that uses filtering so users can directly view their desired food:
  - Cookies
  - Cakes
  - Macaroons

  When the user wants to buy a product, the user can use the add to cart button to add the desired item to cart. From there, the user can press checkout and proceed to the payment page. If the payment is confirmed, the user will get an order number in the confirmation page and will be prompted to go back to the homepage.
  ### Features
  Features that are included in the website are:
  - Contact form
  - In the menu page, users can various activities such as:
    - Filtering
    - Searching
  - Proceed a checkout page with checking system
  - English and Turkish language support so customer can pick between two languages
